# 超爽 der 校務 csv 外掛

## 功能

* 支援表格匯出 csv
* 表格列表增加 1000/2000/3000 列顯示

## 環境需求

需先安裝瀏覽器 Userscript 擴充套件

* Firefox: [violentmonkey](https://addons.mozilla.org/zh-TW/firefox/addon/violentmonkey/)

* Chrome: [tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=zh-TW)
